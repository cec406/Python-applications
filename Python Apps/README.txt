This note taking app will take your notes in line by line if you select option 1. 
When you are done, type done.
If you want to view your notes you can press 2, and if you press 3, you can copy your notes, then paste into AI
   to get a summary, and you can paste them into the program. 
The program will create a txt file in the folder that the program is located in. 
You need colorama installed in python to run this application.
