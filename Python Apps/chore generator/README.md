# Chore Generator

A simple, offline Python tool to randomly generate household chores and tasks. Perfect for families, roommates, or anyone looking to make chore assignment fair, fun, and hassle-free.

Built by **Grizzly System Engineering** ‚Äî reliable Montana IT solutions for home and business.

## Why This Tool?

- Struggling to decide "who does what" around the house?
- Want a quick, unbiased way to create daily/weekly chore lists?
- Need something private, no apps/subscriptions/tracking?

This lightweight script generates randomized or categorized chore suggestions you can run locally in seconds.

## Features

- **Random chore generation** from a built-in list of common household tasks
- **Customizable** ‚Äî easily edit the chore pool in the code
- **Options for**:
  - Number of chores per run
  - Daily vs. weekly mode
  - Assign to people (optional simple input)
- **Console-based** (easy to run in any terminal)
- **100% offline & private** ‚Äî no internet, no data leaves your machine
- Future ideas: GUI version (Tkinter), export to PDF/txt, weighted chores by difficulty

## Requirements

- Python 3.6+ (tested on 3.8‚Äì3.12)
- No external libraries needed (pure Python)
