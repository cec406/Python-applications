# Harambe Escape

A silly, retro-style pixel-art arcade game made with Pygame.

You play as a pixelated Harambe trying to survive in the zoo. Dodge the children ‚Äî touching one summons the zookeeper, who then shoots a bullet straight at you. Survive as long as you can... or don't. It's basically inevitable.

A dark-humor tribute to the 2016 Harambe meme, built as a fun coding experiment by Grizzly System Engineering (East Helena, Montana).

## Gameplay

- Move with arrow keys
- Touch a child ‚Üí zookeeper appears and fires
- Get hit by the bullet ‚Üí Game Over
- After game over: Press Y to play again, N or Q to quit

## Requirements

- Python 3.6+
- Pygame (pip install pygame)

## How to Run

1. Navigate to the folder:
   cd "Python Games/harambe"

2. Install Pygame (one time):
   pip install pygame

3. Launch the game:
   python harambe.py
